# OOP Practice in C++

This repository contains C++ practice programs demonstrating core object-oriented programming concepts.

### Topics Covered
- Classes and objects
- Constructors and destructors
- Encapsulation and abstraction
- Inheritance (single, multilevel, multiple, hierarchical, hybrid)
- Polymorphism
- Static members
- File handling
- Exception handling

## Purpose
Created for learning and practicing object-oriented programming in C++.
